Simulation Analysis

The coherent source beams recombine at the semi-reflective interface (BS-1), inducing interference proportional to the resultant phase differential.

I = 2I₀(1 + cos(δ))

The quantum phase δ is a function of the optical path difference ΔL:

δ = (2π / λ) * 2Δd

Constructive: Maximum intensity achieved when path differential is a multiple of λ.

Destructive: Phase cancellation occurs at half-multiples of λ.
